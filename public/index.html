Importador de Detalles de Operaciones (IOL)

Copia y pega la tabla desde la web de InvertirOnline (Mi Cuenta > Movimientos > Detalles de operaciones).

Hacer clic para seleccionar archivo o arrastrar aquí Soporta el archivo MovimientosHistoricos.xls exportado de IOL
¿Cómo funciona el importador?
  • El parser detectará automáticamente los encabezados o la alineación de columnas.
  • Calcula y limpia símbolos de moneda ($ o USD), puntos y comas de separadores decimales/miles de forma inteligente.
  • Genera un código hash único por fila para **evitar que se inserten filas duplicadas** si vuelves a copiar el mismo rango.
  • La base de datos SQLite física consolidará todas tus compras y ventas de forma definitiva.
